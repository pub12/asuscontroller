# NetWarden — Design

Working design files for NetWarden (ASUS ZenWiFi control app).

## Contents
- `stitch-prompt.md` — the Google Stitch prompt used to generate the screens, plus refinement notes.
- `stitch-screens-review.md` — catalog of the generated screens, mapping to PRD §10, and scope conflicts to resolve.
- `screens/` — exported screen PNGs (see naming in the review doc).
- `netwarden_PRD_v2.md` — current PRD (also lives in `00.lib/`; copied here as the project working copy).
